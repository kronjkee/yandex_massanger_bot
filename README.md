# Бот для Яндекс Мессенджера + Яндекс Трекер

Автоматически находит ключи задач в сообщениях и показывает информацию о них.

## Функции
- 🔍 Поиск ключей формата `PROJECT-123`
- 📊 Показ названия, статуса, исполнителя
- 💬 Ответ в тред к исходному сообщению
- ⚡ Кэширование задач
- 🔐 Работа в нескольких чатах

## Команды
- `/start` — активировать бота в чате
- `/stop` — деактивировать
- `/help` — справка

## Установка
1. Скопируйте `.env.example` в `.env` и заполните токены
2. Запустите: `docker-compose up -d --build`
3. Убедитесь, что вебхук доступен по HTTPS

## API (документация Яндекса)
